# DevOps Portfolio

Este repositorio contiene prácticas y proyectos orientados a DevOps, enfocados en el uso real de herramientas y conceptos clave del ciclo de vida de aplicaciones.

## Tecnologías y conceptos practicados
- Docker (Dockerfile, imágenes, contenedores, PID 1)
- Docker Compose (multi-contenedor)
- Kubernetes (manifiestos básicos, servicios)
- Terraform (Infraestructura como Código)
- CI/CD con GitHub Actions
- Linux y Git

## Estructura del repositorio
Cada carpeta representa una práctica independiente, con su propia documentación y archivos de configuración.

## Objetivo
Demostrar comprensión práctica de entornos containerizados, automatización y despliegues reproducibles, orientado a un rol DevOps Junior.
